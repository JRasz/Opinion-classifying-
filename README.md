Opinion Sentiment Classifier

This project classifies English text as positive or negative based on simple machine-learning techniques. It uses basic text preprocessing and a standard classifier from scikit-learn.

Overview:
-Load and clean the text data
-Convert text into numerical features
-Train a model to classify opinions
-Test the accuracy on new data

This is a beginner-friendly NLP project focused on understanding the basics of sentiment analysis.

What the Notebook Does:

Reads the dataset
-Cleans and preprocesses text (lowercasing, removing symbols, etc.)
-Vectorizes text using simple methods
-Trains a sentiment classification model
-Prints accuracy
-Tests the model on sample inputs

Libraries Used
-pandas
-scikit-learn
